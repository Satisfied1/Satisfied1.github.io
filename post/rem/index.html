rem布局

rem布局优点：等比例放大 rem布局缺点：大屏幕手机上看到元素还是一样多，违反设计原则

rem布局实现方式：

  1. 设计时 按照font-size:20px; 算出了所有的rem。 （可以设置成100，这样计算起来更方便）

  2. 动态的计算出html font-size事件(document.documentElement.clientWidth /设计稿宽度)*20px;

  3. 对于文字大小的处理 设置成px值

    响应式设计 小于400px 14px; 大于400px小于600px 16px; 大于600px 18px;

  4. 字体样式：用手机默认字体

需要用CSS预编译实现rem布局 rem.scss中 $font-size:20;

@function px2rem($px){ @return $px / $font-size * 1rem; } index.scss中 @import “rem”; div{ width: px2rem(100); ) index.html的head中 //先引入jquery，否则下面的js代码用原生写 index.js中，根据屏幕宽度自动按比例更改字体大小 (function() {

var docEl = document.documentElement;
var resizeEvt = 'orientation' in window ? 'orientationchange' : 'resize';

// 在手机端有window.orientation属性,当前设备的屏幕方向，0表示竖屏，正负90表示横屏（向左与向右）模式
var recalc = function() {
        var clientWidth = docEl.clientWidth;

        if (!clientWidth) return;
        //20 ，是scss 定义的变量 就是计算rem的基准
        // 640 是设计图的宽度。
        docEl.style.fontSize = 20 * clientWidth / 640 + 'px';
};

if (!document.addEventListener) return;
window.addEventListener(resizeEvt, recalc, false);

//dom 节点加载完成后就会执行回调函数 DOMContentLoaded
//(img 图片 ，script加载完后 ， dom节点创建完 css，)window.onload
//$(function (){}) == $(document).ready(function(){})页面加载完(dom节点加载完)之后，就会执行这个参数function
document.addEventListener('DOMContentLoaded', recalc, false);

})();
